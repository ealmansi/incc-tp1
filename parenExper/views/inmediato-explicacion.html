Primera parte

Procedimiento

  • Se mostrarán diferentes pantallas en forma sucesiva, presentando en cada una de ellas una secuencia formada por paréntesis ( ), llaves { } y corchetes [ ].
  • Para cada secuencia, usted deberá determinar si los símbolos que en ella aparecen se encuentran correctamente apareados. Más precisamente, decimos que la secuencia se encuentra bien formada si para cada símbolo que abre hay uno opuesto que cierra, y lo que queda encerrado entre el par de símbolos apareado también está bien formado.
  • Luego de cada estímulo, se le presentará una pantalla donde deberá responder la pregunta cliqueando uno de los siguientes botones:
    • Sí     si la secuencia presentada anteriormente se encuentra bien formada.
    • No     si la secuencia presentada anteriormente NO se encuentra bien formada.
  • Cada secuencia será presentada por un total de {{t_expo / 1000 | number:0}} segundos, y luego usted dispondrá de {{t_resp / 1000 | number:0}} segundos para responder.

Esta secuencia está correctamente apareada, ya que todos los términos definidos por los símbolos (denotados por paréntesis y corchetes en este ejemplo) están correctamente cerrados.
