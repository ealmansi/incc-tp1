Procedimiento: primera parte

  • Se mostrarán diferentes pantallas sucesivamente, incluyendo en cada una de ellas una secuencia formada por paréntesis ( ), llaves { } y corchetes [ ].
  • Para cada secuencia, usted deberá determinar si la misma se encuentra bien formada; es decir, si los símbolos están correctamente apareados. Más precisamente, la secuencia está bien formada si para cada símbolo que abre hay uno opuesto que cierra, y lo que queda encerrado entre el par de símbolos apareado también está bien formado.
  • Luego de cada estímulo, se le presentará una pantalla donde deberá responder apretando una de las siguientes teclas:
    • s     si la secuencia presentada anteriormente está correctamente apareada.
    • n     si la secuencia presentada anteriormente NO está correctamente apareada.
  • La secuencia será presentada por un total de {{t_expo}} segundos, y luego usted dispondrá de {{t_resp}} segundos para responder.

Ejemplo 1

( ) ( [ ] )

Este caso está correctamente apareado, ya que todos los términos definidos por los símbolos (denotados por paréntesis y corchetes en este ejemplo) están correctamente cerrados.

Este caso no está correctamente apareado, ya que faltaría cerrar un paréntesis en el segundo término.
