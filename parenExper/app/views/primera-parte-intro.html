Primera parte

El experimento funciona así:

  • Primero, te mostramos una secuencia formada por varios paréntesis ( ), llaves { } y corchetes [ ]. Por ejemplo, esta: ( ) { ( ) [ ] } .
  • Te damos unos segundos para que la mires bien, y después necesitamos que nos digas si pensás que la secuencia está bien o mal formada.
  • Para responder, solo tenés que apretar la letra S (si) o N (no) de tu teclado.

    • Nota: decimos que la secuencia está bien formada si para cada símbolo que abre hay uno opuesto que cierra, y lo que queda encerrado entre uno y otro también está bien formado. Por ejemplo, la secuencia anterior está bien formada, y esta otra no: { [ } ( [ ] ) ] .

    Nota (bis): por las dudas, un par de ejemplos más.
     Esta está bien: { { } } [ ] ( ) .
     Esta está mal: } ( { } ) .
