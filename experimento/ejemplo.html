Explicación experimento

Se mostrarán sucesivamente pantallas incluyendo una única secuencia de símbolos. En cada una de ellas, usted deberá determinar si los símbolos están correcta o incorrectamente apareados.

Ejemplo 1

( ( ) ) ( )

Este caso está correctamente apareado, ya que todos los términos definidos por los símbolos (denotados por paréntesis en este ejemplo) están correctamente cerrados.

Ejemplo 2

( ( ) ) ( ( )

Este caso no está correctamente apareado, ya que faltaría cerrar un paréntesis en el segundo término.

Presione el botón “Continuar” si comprende hasta el momento.
