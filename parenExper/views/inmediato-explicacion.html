Primera parte

Procedimiento

  • Se mostrarán diferentes pantallas en forma sucesiva, presentando en cada una de ellas una secuencia formada por paréntesis ( ), llaves { } y corchetes [ ].
  • Para cada secuencia, usted deberá determinar si los símbolos que en ella aparecen se encuentran correctamente apareados, como se ve en los Ejemplos 1 y 2.
  • Luego de cada estímulo, se le presentará una pantalla donde deberá responder inmediatamente apretando uno de los siguientes botones:
    • s     si la secuencia presentada anteriormente se encuentra bien apareada.
    • n     si la secuencia presentada anteriormente NO se encuentra bien apareada.

Ejemplo 1

( ) ( [ ] )

Esta secuencia está correctamente apareada, ya que todos los términos definidos por los símbolos (denotados por paréntesis y corchetes en este ejemplo) están correctamente cerrados.

Ejemplo 2

{ ( ) } ( ( )

Esta secuencia no está correctamente apareada, ya que faltaría cerrar un paréntesis en el segundo término.

Cuando se encuentre preparado/a, presione 'Comenzar'.
